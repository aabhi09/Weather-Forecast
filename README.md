# Weather Forecast Application
Built by Aabhinav • Bengaluru • 2026

Interactive real-time weather app built with Python & Streamlit.

## Features
- Search any city worldwide
- Current weather, hourly & 7-day forecast
- Air quality (AQI, PM2.5, Ozone)
- Interactive Folium map
- Historical comparison
- Light/dark theme toggle
- PDF report export

## Tech Stack
- Streamlit (UI)
- Open-Meteo API (weather & air quality)
- Pandas, Plotly, Folium, ReportLab

## Run locally
```bash
pip install -r requirements.txt
streamlit run weatherforecasting.py
